Checking rules

After setting all rules for the fields you should apply these rules with the setup() returns void method:

 $this->formbuilder->setup();

Afterwise you can check if all entered values are valid with the validate() returns boolean method:

if($this->formbuilder->validate()){
//use the data from the form
}
//show the form

 If validate() failes it will automatically show the errors in the form.

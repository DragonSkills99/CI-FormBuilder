Create Input

To create an input you have to create an field with the function addField() return Field:

$this->formbuilder->addField();

Now you've an empty Field. If you generate the form there will be an textfield but you can't do anything with it instead of typing text in. To customize the field you have the following options

  • set an attribute (in this example an link was created) setAttribute(name as string, value as string) returns Field:

Field->setAttribute("href", "http://google.com");

  • set the text of the lable in front of the input setDescription(text as string) returns Field:

Field->setDescription("Name:");

  • set the name attribute so the form knows where to put the value setName(name as string) returns Field:

Field->setName("name");

  • set the value attribute so you have an initial value in the input (for textareas use setInnerValue instead) setValue(value as string) returns Field:

Field->setValue("Sebastian Michaelis");

  • insert text beetween the opening and closing tag after all childs setInnerValue(value as string) returns Field:

Field->setInnerValue("Kazuto Kirigaya");

  • set the type of the input (e.g. if you want to only get numbers in this field enter "number") setType(type as string) returns Field:

Field->setType("text");

  • set if the descriptive lable should be shown (true means it won't be shown, false means it will be shown. Standard is false) setFillBothRows(value as boolean) returns Field:

Field->setFillBothRows(false);

  • set the class attribute for styling purposes setClass(value as string) returns Field:

Field->setClass("nameinput");

  • create the submit field setupSubmit(text as string) returns Field:

Field->setupSubmit("Edit name");

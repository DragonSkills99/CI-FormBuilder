Getting started

For using FormBuilder you have to load the library first. You can do this inside of an CI_Controller:

$this->load->library("FormBuilder");
 Or outside:
$this->load = new CI_Loader();
$this->load->library("FormBuilder");
$this->formbuilder = new FormBuilder();

Afterwards the formbuilder is loaded and ready for use.

If you're ready for generating the form just call getForm(). It returns the form as string so you can echo it. (Or overgive it to a view for viewing)

echo $this->formbuilder->getForm();

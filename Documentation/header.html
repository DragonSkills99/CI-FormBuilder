Setting Form Header Data

To set the method simply call setMethod(type as string) returns Formbuilder:

$this->formbuilder->setMethod("POST");

To set the action simply call setAction(action as string) - returns FormBuilder:

$this->formbuilder->setAction("http://google.com");

It is possible to do this as chainaction the order doesn't matter:

$this->formbuilder->setAction("http://google.com")->setMethod("POST");
